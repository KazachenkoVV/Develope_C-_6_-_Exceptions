# Calculator
Исключения

Домашнее задание

Доработайте класс калькулятора способным работать как с целочисленными
так и с дробными числами. (возможно стоит задействовать перегрузку операций)

--------------
Доработать последний вариант каклькулятора, работающий в цикле, не представляется возможным.
Обработка ввода чисел производится в этом цикле поразрядно с испльзованием переменной
типа ConsoleKeyInfo с последующим преобразованием в поле decimal, т.к. заранее не известно
будет ли пользователь вводить десятичную точку или нет. Преобразовывать полученное число в Int,
в случае если точки нет, для того, чтобы в последствии использовать перегрузку методов выглядит абсурдным.
Поэтому для доработки был взят более ранний вариант класса Calculator.
